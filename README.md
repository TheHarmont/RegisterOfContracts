<h1>Реестр контрактов</h1>

<h2>Описание работы</h2>
<p>
  Содержит таблицу с данными контрактов, взятых из <b>zakupki.gov.ru</b>.
  Нажатием на строку, открывается модальное окно с подробной информацией о контракте.
</p>
<p>
  Формирование контракта осуществляется выбором шаблона из выпадающего списка.
  После этого пользователю будет доступна демонстрация содержимого файла, прежде чем он примет решение о том, стоит ли ему скачивать контракт или нет.
</p>
